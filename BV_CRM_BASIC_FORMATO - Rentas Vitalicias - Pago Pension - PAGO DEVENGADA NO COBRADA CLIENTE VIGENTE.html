Instrucciones para solicitar una pensión devengada no cobrada para un cliente vigente

1. Verificar el medio de pago de la pensión, ya que esta opción se aplica a pagos emitidos a través de SERVIPAG, tesorería o custodia tesorería. Esta última se utiliza cuando los beneficiarios deben demostrar su derecho a pensión antes de recibir los pagos devengados y no cobrados.

2. Es importante considerar que el pago debe estar caducado para realizar esta solicitud. Por ejemplo, la pensión del presente mes caduca el último día del mes siguiente, y luego se deben esperar 10 días adicionales para que los pagos sean devueltos a la compañía. Solo entonces se puede solicitar la pensión devengada para el cliente vigente..

3. Si se solicita la pensión en tesorería, el ejecutivo debe asegurarse de que el cheque no esté en la caja.

4. Recuerda que, para las pensiones pagadas vía SERVIPAG, caducan a fin de mes del siguiente mes.

5. Antes de enviar la documentación, asegúrate de digitalizarla y adjuntarla a la solicitud de servicio correspondiente.

6. Es importante notificar al cliente tan pronto como se resuelva la solicitud de servicio, ya que, si el pago vuelve a caducar, el cliente o beneficiario deberá volver a firmar la solicitud para gestionar nuevamente el pago.
